# Provenance Note for Support Artifacts

The attached paper gives aggregate experimental details but does not include raw scenario data, analyst timing sheets, exact prompts, model version, or original logs. The artifacts in this folder are generated to support a transparent reproduction of the reported setup, not to claim recovery of the original authors' private data.

Generated artifacts align with the paper's reported setup:
- 50 action-decision scenarios.
- 2 analysts.
- 100 indirect prompt-injection attempts.
- MTTA targets: B0 = 8.6 seconds, B1 = 6.55 seconds, CACP = 4.9 seconds.
- Mean coordination latency target: 22 ms.
- Modeled LLM inference latency: 1.2 seconds.

Use these files as a reproducibility scaffold. In a paper revision, label them as "reconstructed/synthetic support data" unless the original authors provide the real logs, timing sheets, prompts, and model identifiers.
