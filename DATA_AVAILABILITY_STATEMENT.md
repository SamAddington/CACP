# Data Availability Statement

## Version for original or fresh rerun data

The data and code supporting this paper are available as an artifact package. The package includes the simulated scenario table, analyst timing records, prompt-injection attempts, prompts, model/version manifest, policy-evaluation traces, signed Decision Cards, hash-chained ledger entries, aggregate output files, and checksums. The artifact supports the paper's preliminary evaluation of mean time-to-audit, out-of-policy execution prevention in a bounded indirect prompt-injection suite, and coordination latency.

## Version for reconstructed/synthetic support data

The data and code supporting a reproduction of this paper's reported preliminary evaluation are available as an artifact package. Because the original raw logs, timing records, exact prompts, and model identifiers were not included in the submitted manuscript, the released package provides reconstructed/synthetic support artifacts calibrated to the reported setup and aggregate results. These artifacts should be used to inspect the protocol mechanics and reproduce the reported aggregate behavior, not as a substitute for independently collected analyst timing data.
